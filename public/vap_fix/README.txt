طريقة الاستخدام:

1. قم بتحميل الملفين `vap_solution.py` و `requirements.txt` إلى مجلد واحد على جهاز Mac الخاص بك.
2. افتح الطرفية (Terminal) وانتقل إلى المجلد.
3. ثبت المكتبات المطلوبة:
   pip install -r requirements.txt
4. عدل مسار أداة VAP في ملف `vap_solution.py` (السطر 20).
5. شغل السكريبت لتحويل الفيديو:
   python vap_solution.py video.mp4

هذا السكريبت يحل مشكلة تعليق الأداة على Mac ويضيف نص TOP 6 تلقائياً.
