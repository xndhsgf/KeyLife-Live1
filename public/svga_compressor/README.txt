متطلبات التشغيل

1. تثبيت Python (الإصدار 3.6 أو أحدث).
2. تثبيت مكتبة Pillow للتعامل مع الصور:
   pip install -r requirements.txt
3. تشغيل السكربت:
   python svga_compress.py animation.svga animation_compressed.svga

كيف يعمل الكود؟

1. فك الضغط : ملف SVGA هو في الأصل أرشيف ZIP، يتم فك ضغطه إلى مجلد مؤقت.
2. البحث عن الصور : يبحث في كل المجلدات الفرعية عن ملفات PNG و JPG/JPEG.
3. ضغط الصور :
   · إذا كانت الصورة تحتوي على شفافية (RGBA)، يتم تحويلها إلى لوحة ألوان (P mode) مع الحفاظ على الشفافية وعدد ألوان أقصاه 256 لون. هذا يقلل الحجم بشكل كبير جداً.
   · إذا كانت الصورة بدون شفافية (RGB) وتنسيقها JPEG، يتم حفظها بجودة أقل (قابلة للتعديل).
   · باقي الصور يتم تحويلها إلى لوحة ألوان 256 لون وحفظها كـ PNG مع أعلى مستوى ضغط.
4. إعادة التجميع : يتم إعادة ضغط المجلد المؤقت إلى ملف SVGA جديد.

ملاحظات مهمة

· الحفاظ على الأبعاد : لا يتم تغيير أبعاد الصور مطلقاً، فقط يتم تقليل عمق الألوان وزيادة الضغط.
· تقليل الحجم : يمكن أن يصل تقليل الحجم إلى 50-80% حسب محتوى الصور.
· تعديل الجودة : يمكنك تغيير متغير quality في الدالة compress_svga (القيمة الافتراضية 85) للتحكم بجودة JPEG.
· التوافق : السكربت يحافظ على هيكلية ملف SVGA الأصلية، لذا سيبقى متوافقاً مع أي مشغل SVGA.
